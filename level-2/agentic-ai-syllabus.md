# Agentic Generative AI Developer Program — Level 2 (Weeks 5–8)

Focus: Multi-Agent Systems, Memory & RAG

## Week 5 — RAG Foundations and Azure Agentic Patterns
Weekly objectives:
- Introduce retrieval-augmented generation (RAG) concepts.
- Build agentic workflows with enterprise tools.
- Design an architecture that supports external knowledge.

Assigned courses:
- Course: Develop generative AI apps in Azure (AI-3016) — Link: https://learn.microsoft.com/en-us/training/paths/create-custom-copilots-ai-studio/
- Course: Develop AI agents on Azure — Link: https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-engineer/
- Course: Develop natural language solutions in Azure — Link: https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-engineer/

GenAI topics:
- RAG workflows
- Agentic patterns on Azure
- Tool calling with external data

Labs:
- Build a basic RAG pipeline that retrieves policy snippets.
- Create a multi-step agent flow with a retrieval step and a summarization step.

Deliverables:
- A working RAG prototype with a minimal document set.
- Architecture diagram for the multi-agent flow.

Suggested stack:
- Azure AI Foundry
- Azure AI services
- Python 3.10+

## Week 6 — Multi-Agent Systems with Bedrock
Weekly objectives:
- Implement multi-agent collaboration patterns.
- Use managed GenAI platforms for orchestration.
- Evaluate trade-offs in managed vs. custom agent systems.

Assigned courses:
- Course: AWS Generative AI and AI Agents with Amazon Bedrock (Certificado Prof.) — Link: https://www.coursera.org/professional-certificates/aws-generative-ai-developers
- Course: Lab: Explore Generative AI Use Cases with LangChain and Amazon Bedrock — Link: https://skillbuilder.aws/generative-ai
- Course: AWS Cloud Quest: Generative AI Practitioner — Link: https://aws.amazon.com/blogs/training-and-certification/category/artificial-intelligence/generative-ai/

GenAI topics:
- Multi-agent collaboration
- Amazon Bedrock workflows
- Agent tool usage in managed platforms

Labs:
- Implement a two-agent system for claim verification and counter-argument generation.
- Add a retrieval tool call for external evidence.

Deliverables:
- Multi-agent prototype with at least two specialized roles.
- Short evaluation report on agent collaboration outcomes.

Suggested stack:
- Amazon Bedrock
- Python 3.10+
- LangGraph (concepts)
