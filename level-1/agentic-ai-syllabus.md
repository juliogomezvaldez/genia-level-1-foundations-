# Agentic Generative AI Developer Program — Level 1 (Weeks 1–4)

Focus: Foundations & Single-Agent Systems

## Week 1 — GenAI Fundamentals and Core Concepts
Weekly objectives:
- Establish a solid baseline in generative AI and LLM fundamentals.
- Understand common GenAI use cases and terminology.
- Build vocabulary for prompt-based workflows.

Assigned courses:
- Course: Generative AI for Beginners — Link: https://learn.microsoft.com/en-us/shows/generative-ai-for-beginners/
- Course: Fundamentals of Generative AI — Link: https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/
- Course: OpenAI Academy Knowledge Hub — Link: https://academy.openai.com/

GenAI topics:
- Generative AI fundamentals
- Large Language Models (LLMs)
- Core GenAI use cases

Labs:
- Summarize a short article using a basic prompt template.
- Compare two prompt styles for the same task and record differences.

Deliverables:
- A 1–2 page learning brief with key concepts and examples.
- A simple prompt library with at least 5 reusable prompts.

Suggested stack:
- Python 3.10+
- OpenAI API

## Week 2 — Prompt Engineering and OpenAI API Basics
Weekly objectives:
- Apply prompt engineering patterns to real tasks.
- Learn basic API usage for generation and transformation.
- Produce structured outputs for downstream use.

Assigned courses:
- Course: ChatGPT Prompt Engineering for Developers — Link: https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/
- Course: Documentacion de la API de OpenAI y Tutoriales — Link: https://platform.openai.com/docs/overview
- Course: DataCamp: 'Become an AI Developer' Code Along Series — Link: pending – to be provided by the user

GenAI topics:
- Prompt engineering patterns
- Structured outputs
- OpenAI API usage

Labs:
- Build a prompt that extracts factual claims from a paragraph.
- Call the OpenAI API to summarize and reformat text.

Deliverables:
- A small script that runs 3 prompt patterns against sample inputs.
- A JSON schema for model outputs.

Suggested stack:
- Python 3.10+
- OpenAI API

## Week 3 — Productivity Copilots and Tooling
Weekly objectives:
- Explore developer copilots and their capabilities.
- Practice integration of AI assistants in a coding workflow.
- Identify limitations and safe usage patterns.

Assigned courses:
- Course: Get Started with GitHub Copilot — Link: https://learn.microsoft.com/en-us/training/modules/get-started-github-copilot/
- Course: PartyRock — Link: https://aws.amazon.com/ai/learn/
- Course: Amazon CodeWhisperer - Getting Started — Link: https://aws.amazon.com/startups/learn/choosing-the-right-generative-ai-learning-path

GenAI topics:
- Coding assistants and copilots
- Prompt-assisted development
- Practical usage constraints

Labs:
- Use a copilot tool to scaffold a small CLI that reads and writes files.
- Compare manual vs. copilot-assisted refactors.

Deliverables:
- A short report on productivity gains and risks.
- A mini-project scaffold generated with a copilot tool.

Suggested stack:
- VS Code
- GitHub Copilot or Amazon CodeWhisperer

## Week 4 — Intro to Agents (Single-Agent)
Weekly objectives:
- Understand agentic workflows at a high level.
- Build a simple single-agent loop with tools.
- Prepare for multi-agent and RAG in Level 2.

Assigned courses:
- Course: Copilot Studio Agent Academy — Link: https://learn.microsoft.com/en-us/copilot/
- Course: Foundation: Introduction to LangGraph — Link: https://academy.langchain.com/courses/intro-to-langgraph?utm_source=chatgpt.com
- Course: Developing gen AI Applications on AWS — Link: https://aws.amazon.com/training/classroom/

GenAI topics:
- Single-agent workflows
- Tool calling and orchestration basics
- Inference parameters and prompt iteration

Labs:
- Create a single-agent loop that classifies claims by confidence.
- Add a basic tool call step for web-style lookup simulation.

Deliverables:
- A working single-agent prototype with tool call stubs.
- A short design doc describing agent inputs, outputs, and steps.

Suggested stack:
- Python 3.10+
- OpenAI API or Amazon Bedrock
- LangGraph (concepts)
