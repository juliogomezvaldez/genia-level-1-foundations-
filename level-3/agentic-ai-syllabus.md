Weekly objectives:
- Optimize LLM usage for latency, cost, and reliability.
- Introduce evaluation, monitoring, and testing workflows.
- Harden prompt and output contracts for production.

- Course: OpenAI for Developers (Learning Path) — Link: https://www.pluralsight.com/paths/openai-for-developers
- Course: OpenAI GPT API (Aspire Journey) — Link: https://www.skillsoft.com/journey/openai-gpt-api-7999e9dc-6ed2-4e28-98e7-478a6b72436b
- Course: OpenAI API Complete Guide: With Practical Examples in Python — Link: pending – to be provided by the user

GenAI topics:
- LLM latency, cost, and limits
- Testing, monitoring, and evaluation
- API reliability patterns

Labs:

- Add automated evaluation checks for output quality.

Deliverables:
- A performance and cost report for multiple model configurations.
- A minimal evaluation harness with test cases.
